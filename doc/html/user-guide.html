Archivarius user guide

Archiving

Archive can be created in several ways
1. From file system, select necessary files and directories in file list, press button Archive in toolbar, menu or press key F5.
2. From file system or inside archive, drag files from explorer into program window.
3. Paste files from clipboard (Ctrl + V).
Then screen will show a window with compression settings.
If archive exists, it will be opened for appending files.
Update compressed archive can be very long.
First need to unpack all files, and then archiving.

Unarchiving

There are several ways to extract files from the archive
1. From file system, select necessary archives in file list, press button Unarchive in toolbar, menu or press key F6.
2. Inside archive select necessary files and directories in file list to unpack, press button Unarchive.
3. Copy files to clipboard (Ctrl + C). Then paste to explorer (Ctrl + V).
Then screen will show a window with uncompression settings (method 1 and 2).
Before copying files to clipboard, they will be unpacked to temp directory.
If data has been encrypted with a password, corresponding window will be displayed before unpacking.

Testing

Archive testing is intended to verify data integrity.
Test archives can also be from the file system.
Press button Test in toolbar, menu or press key F7.

Information

From file system displays information about files, directories and archives.
Inside archive displays information about current opened archive.
Press button Information in toolbar, menu or press key F8.
Then screen will show a window with information.

Deletion

Files and directories can be deleted both from file system and from archive.
Press button Delete in toolbar, menu or press key delete on keyboard.
Deleting files from a compressed archive can be very long.
First need to unpack all files, and then archiving.

Renaming

You can rename files and directories from file system as well as from archive.
Press button Rename in menu or press key F2 on keyboard.
