Archive information


Archive information window consist of fourteen labels and one button.

Labels

Item Description
Archive File name.
Files count Inside archive.
Source size Source files size before compression.
Compress size Compression size.
Ratio Compression ratio.
Encryption Encryption state of files.
Preprocess Bigraph preprocessing state of files.
Method Compression method.
Level Compression level.
Dictionary Dictionary size.
Match Minimum and maximum search match length.
Threads Compression threads number.
