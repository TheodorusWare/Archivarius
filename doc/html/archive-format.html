Archivarius

Archive file format

Archive header 40 bytes

Offset Bytes Description
0x00 4 Archive signature arv3, 0x33767261 (little endian).
0x04 4 Count files in archive.
0x08 8 Files data size.
0x10 8 Preprocessing bigraph size.
0x18 4 Compression properties.
0x1C 4 Archive options, low 2 bits encryption.
0x20 4 Crc-32 crypt string.
0x24 4 Crc-32 archive header.

File header 22 bytes

Offset Bytes Description
0x00 2 File name length.
0x0A 8 File data size, uncompressed.
0x0E 4 File data crc-32.
0x0A 8 Last modified time for os windows.

Archive file structure

Item Description
1 Archive header.
2 All files headers.
3 All files names.
4 All files datas.
