Archivarius interface

Compression settings window


Compression settings window consist of three buttons, three checkbox, three edit fields, six combobox and nine labels.

Buttons

Item Description
... Archive path over file dialog.
OK Close window and start compression.
Cancel Close window and return to main window.

Edit fields

Item Description
Archive Destination path.
Parameters Command line parameters for advanced compression control.
Password Protect your data.

Checkbox

Item Description
Bigraph preprocessing Improvement compression ratio.
Encrypt file names Advanced protect your data.
Password show Verify password is correct.

Combobox

Item Description
Compression level From Store (without compression) to Ultra (best settings).
Compression method LZRE or CM.
Dictionary size LZRE method possible to set size in range [64k, 256m].
CM method possible to set size in range [64k, 16m].
Match length maximum Maximum match length for LZRE in range [32, 1024]. Context length for CM in range [4, 8].
Match length minimum For LZRE method. Possible range [2, 8]. Minimum length [4, 8].
Length less 4 sets prefix length for hashing. Default value 5.
Threads number For LZRE method. Possible range [1, 2].
